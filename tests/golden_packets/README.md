# Golden Packets

Golden packets are minimal, reproducible debug cases used to prevent regression in the workbench. Each case contains:

- `README.symptom.md`: symptom summary and trigger.
- `debug_packet.yaml`: structured evidence packet.
- `expected_report.md`: expected report shape or key conclusions.

These packets are intentionally small mocks. Replace or extend them with real field cases after a root cause has been verified.
