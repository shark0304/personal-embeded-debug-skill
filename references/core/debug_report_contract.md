# Debug Report Contract

Use this structure when producing a debug report or final triage report.

## Required Sections

1. Case summary
2. Evidence completeness
3. Missing critical evidence
4. Hypothesis ranking
5. Verification plan
6. Fix plan
7. Regression recommendation

## Report Rules

- Keep facts and hypotheses separate.
- Every hypothesis must include a verification step.
- Every fix must say what evidence would justify applying it.
- Every regression recommendation must name the artifact or test to preserve.
- If the packet is incomplete, the report must say which conclusion cannot yet be made.
